# 🧟 Zombie Survival – Jungle Shooter

## 📌 Project Overview

Zombie Survival is a **3D jungle survival shooter game** where the player must survive against zombies roaming across the jungle map. The player is equipped with a gun and must eliminate zombies before losing all health.

The game environment is set in a **jungle area containing village houses, ruins, and fencing**, creating a survival atmosphere where zombies appear throughout the map.

The goal of the game is simple: **survive as long as possible while killing zombies.**

---

## 🎮 Game Features

### 🌴 Open Jungle Map
The player can freely explore the jungle environment, which includes houses, ruins, and fenced areas.

### 🔫 Player Weapon System
- Player starts with a **gun**
- Gun contains **32 bullets (ammo)**
- The player must manage ammo carefully while fighting zombies

### ❤️ Player Health System
- Player starts with **100 HP**
- Zombie attacks **reduce player health by 2 HP per hit**

### 🧟 Zombie Enemies
- Zombies are spread across the jungle map
- Zombies **chase and attack the player**
- Each zombie requires **2 bullets to be killed**

### 🕹 Player Movement
The player can move freely around the entire map to explore and survive while avoiding zombie attacks.

## 🎯 Gameplay

1. The player spawns in a **jungle environment**.
2. Zombies are already present across the map.
3. The player must **explore the environment and shoot zombies**.
4. Each zombie requires **two bullets to be killed**.
5. If a zombie reaches the player, it **attacks and reduces player health**.
6. The player survives until **health reaches zero**.
